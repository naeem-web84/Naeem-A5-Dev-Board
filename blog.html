Question-1:What are the different ways to select an elements in a DOM
In JavaScript DOM is the short form of Document Object Model. In JS it's a tree like structure. However there are several ways to select the elements in a DOM. But there are Five standards way to select the elements. They are :
1. Selects an elements by it's ID = getElementById();
2.Selects an elements by it's className = getElementsByClasssName();
3.Selects elements by their tag name = getElementsByTagName();
4.Using CSS selectors select the first element = querySelector();
5.Select all matching elements using CSS selector = querySelectorAll();
This five is enough to select any elements in a dom eventhough there are some alternate selectors like, document.forms, document.image etc.

Question-3: What is event delegation in the DOM?
Event delegation is a technique which works by Javascript Event Bubbling. where we target the parent element , if anything happens with its child it can understand some thing is done in my child. In simplest way Event delegation is a technique where we set an event in parent So that when an event occurs in the child elements, it can be captured.
Question-4: What is event bubbling in the DOM?
Event Bubbling is a default behaviour of JavaScripts. It's a process where at first an event occours in a child element and then that event climb up in its parent and then its grandParent then in its root element. Its travels in upward like a bubble, like a fully filled glass of water where a bubble comes from its bottom to top. So its called Bubbling , because the event bubble up to the parents .
